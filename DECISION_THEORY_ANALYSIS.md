# Анализ теории принятия решений для ИИ-помощника

## Ключевые выводы для OpenManus RL

### 1. Математические основы оптимальных решений
- **Ожидаемая полезность**: максимизация EU(d) = ∫ p(o|d) · U(o) do
- **Уравнение Беллмана**: V(s) = maxₐ [R(s,a) + γ∑ₛ' P(s'|s,a) V(s')]
- **Принцип оптимальности**: оптимальная стратегия обладает рекурсивным свойством
- **Правило 37%**: оптимальная остановка при последовательном выборе

### 2. Алгоритмы оптимизации
- **MDP и RL**: поиск оптимальной политики π*(s) = arg maxₐ Q*∞(s, a)
- **Q-обучение**: оценка ценности пар (состояние, действие)
- **Policy Gradient**: прямая оптимизация параметрической политики
- **MCTS**: разведка возможных ходов + нейронные сети

### 3. Многокритериальные подходы
- **Парето-оптимальность**: недоминируемые решения
- **MAUT**: U(aᵢ) = Σ wⱼ · uⱼ(xᵢⱼ)
- **AHP и TOPSIS**: иерархическая структуризация и ранжирование
- **NSGA-II**: эволюционный алгоритм для Парето-фронта

### 4. Поведенческие аспекты
- **Ограниченная рациональность**: сатисфикация вместо оптимизации
- **Теория перспектив**: оценка от точки отсчёта, неприятие потерь
- **Минимакс-сожаление**: минимизация упущенной выгоды
- **Размытая логика**: решения в условиях нечёткости

### 5. Практические алгоритмы
- **A***: оптимальный путь с эвристикой
- **Деревья решений**: иерархические правила "если-то"
- **Случайные лесы**: ансамблирование для снижения дисперсии
- **RLHF**: обучение на основе обратной связи от людей
