# Спринт 1: Интеграция фреймворка принятия решений

## Цель
Полная интеграция фреймворка принятия решений в основной проект OpenManus

## Задачи
1. Создать модуль интеграции в app/agent/manus.py
2. Добавить конфигурацию принятия решений в config/config.toml
3. Реализовать выбор действий на основе фреймворка
4. Добавить логирование принятия решений
5. Создать тесты интеграции

## Результат
OpenManus использует фреймворк принятия решений для выбора действий
